[ToasterFX](../../../toasterfx) 的演示项目。

## 演示内容

* 内置类型的消息展示
* 有/无标题、有/无进度、有/无音效、有/无关闭的消息展示
* 自定义类型、自定义音效、自定义FXML
* 超长消息、事件消息、批量消息、并发消息
* 扩展国际化语言
* 切换主题、切换弹窗方式
